👋 Hallo CLI Tool

Ein simples CLI-Tool, das dich in verschiedenen Sprachen begrüßt!

📦 Installation

npm install -g hallo_bern6666

▶️ Verwendung

hallo --name Max --language en

Ausgabe:

Hello Max!

⚙️ Optionen

  • --name – Gibt deinen Namen an
  • --language – Unterstützte Sprachen: de, en, es, fr
  • --version – Zeigt die aktuelle Version an
  • --help – Zeigt diese Hilfe an
